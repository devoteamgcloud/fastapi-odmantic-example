# FastAPI + ODMantic

## Stack

- [FastAPI](https://fastapi.tiangolo.com/)
- [ODMantic](https://art049.github.io/odmantic/)
- [Poetry](https://python-poetry.org/)

## Run app

```bash
docker-compose up
```

## Stop app

```bash
docker-compose down
```
